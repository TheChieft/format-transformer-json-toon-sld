## How to Add Screenshots

1. Take screenshots of your running application
2. Save them in this `docs/` folder
3. Update the main README.md to reference them:
   ```markdown
   ![Main Interface](./docs/hero-screenshot.png)
   ```

## Tools for Screenshots

- macOS: Cmd+Shift+4
- Windows: Win+Shift+S
- Linux: Screenshot tool (varies by distro)
- Browser DevTools: Responsive mode for mobile screenshots

## Tips for Great Screenshots

- Use a clean dataset example
- Show the UI in both light and dark modes
- Capture the metrics panel with data
- Include mobile responsive views
- Use consistent browser window size
- Remove any personal information
